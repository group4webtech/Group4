INSTALLING THE WEBSITE
- install node Js from https://nodejs.org/en/
- install python from https://www.python.org/downloads/
- install all the node packages with:
npm install express, fs, python-shell, child_process, uniqid
- install all the python packages: 
python -m pip install --upgrade pip, pandas, bokeh, networkx, holoviews, maplotlib, sklearn, bs4, sys, json, os
- then go the directory where you have the v2 folder downloaded in your cmd with:
cd YOUR PATH/v2
- and then type "nodemon" in the cmd the run the localhost
- go to "localhost:3000" to view the website

USING THE WEBSITE
- to upload a csv file go to upload file -> choose file -> pick your file -> upload
- then pick which visualization you want to see
